# Alberto Scalia – Personal Portfolio

This repository features the source code of my personal portfolio, built to demonstrate the synergy between **minimalist design** and **modern frontend performance**. Originally developed as a milestone project for the **start2impact** HTML & CSS course, it has been evolved into a sophisticated Vue.js application.

## 🚀 Technical Stack & Features

The project is engineered for speed and maintainability, reflected in the codebase composition:

* **Vue.js Framework**: Powers the core architecture, ensuring a reactive and component-driven user experience.
* **Advanced Styling (SCSS)**: A modular CSS architecture using variables for the "Deep Dark" theme and precise layout control.
* **SEO & Semantic HTML**: Structured to ensure maximum discoverability and accessibility.
* **Automated Deployment**: Integrated with **GitHub Pages** for seamless CI/CD and instant updates.
* **EmailJS**: Client-side contact form management, removing the need for a complex backend.
* **Performance Optimized**: Minimized assets and clean code practices for rapid load times.

## 🎨 Design Philosophy: Visual Synthesis

As an Art Director and Growth Marketer, I designed this interface to focus on **conversion through clarity**:

1.  **High-Contrast UI**: A deep black foundation paired with strategic accent colors to direct user attention.
2.  **Typographic Dominance**: Bold sans-serif fonts used to establish an immediate professional hierarchy.
3.  **Efficiency**: A distraction-free UX designed to lead visitors straight to the work that matters.
